INI 檔案教學

▶ INI 檔案結構說明

典型的 INI 檔案格式：

[SectionName]
Key1=Value1
Key2=Value2

每個 INI 檔包含多個部分（sections），以方括號包圍的部分名稱來區分，內部由鍵值對（key-value pair）組成。這些檔案通常用於設定應用程式的參數，比如使用者偏好、視窗佈局、資料庫連線等。

▶ INI 檔案內容範例

範例內容如下：

[UI Settings] tabPage1.BackColor=-2302756 tabPage1.ForeColor=-16777216 this.BackColor=-986896 tabPage.BackColor=-2302756 tabPage.ForeColor=-16777216 tabControl1.BackColor=-2302756 tabControl1.ForeColor=-16777216 [FontSettings] FontFamily=新細明體 Size=9 Style=Regular

▶ INI 寫入範例

建立 IniFile 物件

private IniFile iniFile = new IniFile(
  Path.Combine(AppDomain.CurrentDomain.BaseDirectory, "ini", "config.ini")
);

寫入設定值

iniFile.Write("UI Settings", "tabPage1.BackColor", tabPage1.BackColor.ToArgb().ToString());
iniFile.Write("UI Settings", "tabPage1.ForeColor", tabPage1.ForeColor.ToArgb().ToString());
iniFile.Write("UI Settings", "this.BackColor", this.BackColor.ToArgb().ToString());

iniFile.Write("FontSettings", "FontFamily", Font.FontFamily.Name);
iniFile.Write("FontSettings", "Size", Font.Size.ToString());
iniFile.Write("FontSettings", "Style", Font.Style.ToString());

▶ INI 讀取範例

tabPage.BackColor = Color.FromArgb(int.Parse(
  iniFile.Read("UI Settings", "tabPage1.BackColor")
));
tabPage.ForeColor = Color.FromArgb(int.Parse(
  iniFile.Read("UI Settings", "tabPage1.ForeColor")
));
this.BackColor = Color.FromArgb(int.Parse(
  iniFile.Read("UI Settings", "this.BackColor")
));

this.Font = new Font(
  iniFile.Read("FontSettings", "FontFamily"),
  float.Parse(iniFile.Read("FontSettings", "Size")),
  (FontStyle)Enum.Parse(typeof(FontStyle), iniFile.Read("FontSettings", "Style"))
);
