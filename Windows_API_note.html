Windows API 教學筆記

▶ 什麼是 Windows API？

Windows API（Windows Application Programming Interface）是 Microsoft 提供的一組函式和服務，讓開發人員能夠直接與 Windows 作業系統進行交互，執行各種低層級操作，如檔案管理、記憶體分配、圖形渲染、系統設定等。

這些 API 是 Windows 系統的核心組件，許多應用程式和軟體都使用它們來完成系統級操作。

▶ Windows API 的主要功能

系統管理（Kernel32.dll）

  • CreateFile：建立或開啟檔案
  • ReadFile / WriteFile：讀取與寫入檔案
  • GetPrivateProfileString / WritePrivateProfileString：讀取與寫入 INI 設定檔
  • Sleep：讓執行緒暫停指定的毫秒數

視窗管理（User32.dll）

  • MessageBox：顯示訊息框
  • FindWindow：查找應用程式窗口
  • SetWindowText：改變窗口標題
  • GetAsyncKeyState：檢測按鍵是否被按下

圖形渲染（Gdi32.dll）

  • CreateCompatibleDC：建立虛擬的畫布
  • BitBlt：繪製位圖到視窗或圖像
  • StretchBlt：縮放位圖

網路通信（Ws2_32.dll）

  • socket：建立網路連接
  • connect：與遠端伺服器建立 TCP 連線
  • send / recv：發送或接收網路數據

▶ 如何在 C# 使用 Windows API

在 C# 中，你可以使用 DllImport 來調用 Windows API。例如，調用 MessageBox：

using System.Runtime.InteropServices;

class Program
{
    [DllImport("user32.dll")]
    public static extern int MessageBox(IntPtr hWnd, string lpText, string lpCaption, uint uType);

    static void Main()
    {
        MessageBox(IntPtr.Zero, "Hello, Windows API!", "Message Box", 0);
    }
}

這段程式碼將顯示一個 Windows 訊息框，標題為 "Message Box"，內容為 "Hello, Windows API!"。

▶ Windows API 的重要性

  • 低層級系統管理工具（磁碟管理、記憶體監控）
  • 硬體驅動程式（如滑鼠、鍵盤、顯示器）
  • 高效能應用（影像處理、遊戲引擎）
  • 企業應用（自動化、檔案管理）
