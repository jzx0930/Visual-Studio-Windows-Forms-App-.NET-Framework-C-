🛠️ 安裝 IDE：Visual Studio Code

安裝完 VSCode 後，請至「套件擴充區」安裝以下插件：

  1. Chinese (Traditional) Language Pack for Visual Studio Code
  2. GitHub Codespaces
  3. GitHub Copilot
  4. GitHub Copilot Chat
  5. Jupyter
  6. Opencv Snippets
  7. Pylance
  8. Python
  9. Python Debugger
  10. Python Environments
  11. Python Image Preview
  12. View Image for Python Debugging

🐍 安裝 Python 主程式

請至 Python 官網 下載最新版本並安裝。

📦 pip 套件管理

安裝完成後，可使用cmd輸入 pip list 查看已安裝的套件。

✅ 建議安裝的套件

  • pip install opencv-python
  • pip install opencv-contrib-python

🧩 待安裝（視需求）

  • pip install pygetwindow
